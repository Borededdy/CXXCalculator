# cxx-calculator - made by @Borededdy

Cxx Calculator is a multipurpose calculator made in C++.
Original inspiration by C-Calculator.

Functions:
• Multilanguage (italiano, english)
• Smart, Fast, Simple
• 4 operators support

General math functions:
• Sum
• Subtraction
• Moltiplication
• Division
